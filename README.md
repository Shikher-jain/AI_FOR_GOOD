AI-Assisted Housing Inspection using Snowflake

This project detects hidden building defects using AI on inspection notes
and image metadata, assigns risk scores, and generates plain-language
inspection summaries for buyers and regulators.

Live Demo: https://ai-housing-inspection-sj-09.streamlit.app/

Tech Stack:
- Snowflake
- Snowflake Cortex (AI SQL)
- Snowpark Python
- Streamlit
